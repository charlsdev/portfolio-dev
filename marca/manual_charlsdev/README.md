## Uso rápido

- **Logo principal**: el monograma (carpeta `logos`). Por debajo de 28px, usar una versión avatar.
- **Logo con nombre**: el wordmark (carpeta `wordmark`). Versión oficial = composición horizontal.
- **Favicon del sitio**: `logos/favicon.svg` con fallback a los PNG.
- **Desarrollo**: `DESIGN_TOKENS.md` tiene colores, tipografía, radios y config de Tailwind lista para copiar.

## Resumen del sistema

- **Logo**: monograma — la C de Carlos abraza los brackets `< >` del símbolo de código original.
- **Color primario**: Signal Amber `#F5B544` · **Acento**: Logic Cyan `#4DD4E0`
- **Tipografía**: Space Grotesk (títulos) · Inter (cuerpo) · JetBrains Mono (código)

Todos los detalles, reglas de uso y aplicaciones están en el manual PDF.
